Graphs

A graph is a pictorial representation of a set of objects where some pairs of objects are connected by links. The interconnected objects are represented by points termed as vertices, and the links that connect the vertices are called edges.
Lets consider a graph as a pair of sets (V, E), where V is the set of vertices and E is the set of edges connecting the pairs of vertices. Take a look at the following graph −

In the above graph, V = {a, b, c, d, e}
E = {ab, ac, bd, cd, de}

Graph representantion

This can be done in many ways but the most common ways are:

  • Adjacency Matrix: A structure for representing a graph in which the presence of arcs between nodes is indicated by an entry in a matrix.
  • Adjacency Lists: An alternative structure for representing a graph in which the arcs are stored as lists of connections between nodes.

The following are the basic operations of the graph structure:

  • Add Vertex − Adds a vertex to the graph.
  • Add Edge − Adds an edge between the two vertices of the graph.
  • Display Vertex - Displays the vertex of the graph.
