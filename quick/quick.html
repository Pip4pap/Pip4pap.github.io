Quick sort

Quicksort is a divide and conquer algorithm. It is a highly efficient sorting algorithm and is based on partitioning of an array into smaller arrays using a pivot element. An array to be sorted is partitioned into two parts one of which holds values smaller than the pivot value, based on which the partition is made and another part of the array holds values greater than the pivot value.
Quick sort partitions an array and then calls itself recursively to sort the two resulting subarrays. This sorting algorithm is quite efficient for large data sets as its average and worst case are O(n2) where n is the input size.

Picking a pivot

A quick sort first selects a value, which is called the pivot value. Although there are many different ways to choose the pivot value, we will simply use the first item in the list. The role of the pivot value is to assist with splitting the list.

Partitioning the array

Partitioning begins by locating two position markers ie call them leftmark and rightmark at the beginning and end of the remaining items in the array (ignoring the pivot element). The goal of the partition process is to move items that are on the wrong side with respect to the pivot value while also converging.

Algorithm

    Step 1 − Choose the highest index value has pivot
    Step 2 − Take two variables to point left and right of the list excluding pivot
    Step 3 − left points to the low index
    Step 4 − right points to the high
    Step 5 − while value at left is less than pivot move right
    Step 6 − while value at right is greater than pivot move left
    Step 7 − if both step 5 and step 6 does not match swap left and right
    Step 8 − if left ≥ right, the point where they met is new pivot

We begin by incrementing leftmark until we locate a value that is greater than the pivot value. We then decrement rightmark. At the point where rightmark becomes less than leftmark, we stop. The position of rightmark is now the split point. The pivot value can be exchanged with the contents of the split point and the pivot value is now in place. Note that all items on the left of the split point are less than the pivot value, and all the items to the right of the split point are greater than the pivot value. The array can now be divided at the split point and the quick sort can be invoked recursively on the two halves.
