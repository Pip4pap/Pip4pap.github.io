Algorithms

An algorithm is a step-by-step procedure for involving a list of instructions that have to be executed in order to accomplish a desired task.
Algorithms are designed to accomplish tasks that include searching, insertion, deletion, sorting and merging data. Some types of algorithms are listed below:

  • Simple recursive algorithms
  • Backtracking algorithms
  • Dynamic programming algorithms
  • Greedy algorithms
  • Divide and conquer algorithms
  • Branch and bound algorithms
  • Randomized algorithms
  • Brute force algorithms

Algorithms are a very important aspect when it comes to the efficiency of computer programs i.e. two programs may look very similar and are built to solve the same problem but they are distinct in terms of how they use computing resources(execution time and memory) in order to solve the problems they are designed to solve.This is all determined by the algorithm that was used to write the program hence it is always important to write algoritms that will maximise utility by using the very least amount of computing resources as posible.Therefore one algorithm can be determined to be better than the other because it is more efficient in terms of the computing resouces used.
