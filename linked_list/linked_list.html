Linked lists

A linked list is a linear data structure where each element is a separate object(Node). Each element of the list comprises of two items;

  • Data
  • Reference of the next node

The entry point into the linked list is called the Head whereas the the last node is always referenced to Null. It is a dynamic data structure;i.e the number of nodes is not fixed implying it can grow and shrink on demand of the user. Linked lists are grouped into two;

  • Singly Linked lists
  • Doubly Linked lists: it has two references(one to the next node and the other to the previous node).

The operations in the linked lists include:

  • Enqueue: this adds an item to the linked lists.
  • Dequeue: this removes an item from the linked list
  • Isempty: is a function to check whether linked list is empty
  • Isfull: this function returns true if linked list is full and false otherwise
  • front: returns the front item of the linked list

Real life examples

There are many real life examples of linked lists. A few are listed below:

  • Train: A train starts from the engine then the coaches follow. You can traverse one coach to the next if they are connected.
  • Steel chain
  • Hunt for problem games
